Popup Menus

All JTextFields have the following Popup Menuitems: Undo, Cut, Copy, Paste, Select All, Clear. And all JTextAreas have the following Popup Menuitems: Undo, Redo, Cut, Copy, Paste, Select All, Clear. And the Go Talking Area has other three popup menuitems:

Send

The "send" button or menuitem has two functions:

Firstly, if there are any selections in the Go Talking Area, this menuitem (and the button) will be activated. Otherwise, it will be disabled. When the user choose this operation, the system will send the words selected by the user to his/her partner via the network and the menuitem will be disabled again.

Secondly, if the user has just recorded a piece of voice and ended the recording, the "send" button and menuitem will be activated. When this operation is choosed, the voice will be send to his/her partner via the network.

Useful Words

There are some subitems of this menuitem. They are all words which maybe useful to the user and they are used to save the user's input time. You can configure them before you use them. When you choose a subitem of this, the words will be pasted to the Go Talking Area without sending them out.

Color Configuration

You can configure the color used in the Go Talking Area. A table will colors will be pop up while you choose this operation. What you set will be saved. If you have changed the words' color and choose undo to restore the words' color, the system cannot replace the color correctly.
