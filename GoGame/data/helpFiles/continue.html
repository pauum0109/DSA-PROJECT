Continue A Game

You can choose to continue a game at any time if the server and your partner's program are both working. This operation will ask you whether you want to save the game when there are any changes happen in you current game and ask for your partner's agree. And only if your partner agrees to continue a game, will a game be continued. If all are OK, an open dialog will be shown to let you to select a go file to open. The file will be open in your part when you confirm the selection and it will also be sent to your partner via the network. Then the system will tell you who is the next turn.
