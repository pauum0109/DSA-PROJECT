About The Go World (Net Version)

This is the Network Version of the Go World Game and it was designed for the friends who like the go game and who are physically separated but can still be connected on the network. You and your partner can play the go game via this software on a net using TCP/IP protocol. So, this software try to act friendly to the players and, give an easy way for you and your friend to treat each other friendly.

I hope that you will like this game. Should you find any bug or have any suggestions, please do let me know.
