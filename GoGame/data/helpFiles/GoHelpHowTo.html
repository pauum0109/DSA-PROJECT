Go World (Net Version)

This is the Net Version of the Go World Game and it was designed for the friends who like the go game and who are separated by the living places but connected by the network. You and your partner can play the go game via this software on a net using TCP/IP protocol. So, this software try to act friendly to the players and, give an easy way for you and your friend to treat each other friendly.

This is the main help file of this game. Hope it will help you. I also hope that you will like this game. If you find any bug or have any suggestions, do let me know. See the about this game part to contact me.
